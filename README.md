## NACL-DEV
Web app for developers to submit there games in Nacl-Appstore.
### Development

#### Framework/libraries used are 
1. Codeigniter
2. PHPMailer
3. SQNile (Own library for database queries)

#### Database Tables
Database name should be nacl with table names as follows:
1.developer 
2.files
3.games
4.users
